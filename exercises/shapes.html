Circle

The cx and cy attributes define the x and y coordinates of the center of the circle.

If cx and cy are omitted, the circle's center is set to (0,0).

The r attribute defines the radius of the circle.

Ellipse, similar to Circle, but has rx and ry radius

Rectangle

Use opacity:0.5 to set opacity value for the whole element, legal range is 0 to 1

The rx and ry attributes rounds the corners of the rectangle

Line

Polyline

Polygon, the points attribute defines the x and y coordinates for each corner

Polygon Star with fill-rule:nonzero AND fill-rule:evenodd

Path, each letter has a specific code

Text with rotation
