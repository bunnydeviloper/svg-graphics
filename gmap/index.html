Sophia's First Google Map

  1. Set the map coordinates to your home town with normal zoom view
  2. Add map overlay (= marker/object on the map that are bound to latitude/longtitude coordinates)
  3. Add map circle around google building
  4. Add a path from San Jose to Big Basin, then to Santa Cruz
  5. Attach event handlers: click the marker to zoom in, then zoom out again after 3 seconds
  6. Marker onclick should also show and info window say "Hello World!"
  7. Add placeMarker() function for every new click and show an infowindow w/ the latitudes and longitudes of the spot
  8. Modify map options, controls, and types
  9. If you zoom really close (scale of 20m), you'll see 90 deg rotate button and 45 deg tilt
