Sophia's First Google Map

  1. Set the map coordinates to your home town
  2. Add Google Mapps Overlay (= objects on the map that are bound to latitude/longtitude coordinates
  3. Add a path from San Jose to Big Basin, then to Santa Cruz
  4. Attach event handlers: click marker to zoom in, then zoom out again after 3 seconds
  5. Add a random circle and an info window at marker
  6. Add placeMarker() function which places a marker where the user has clicked, and shows an infowindow with the latitudes and longitudes of the marker
